## 产品概述

### 产品定位

一站式学语言助手是一款基于"可理解输入"理论的沉浸式语言学习产品，通过浏览器扩展、云端服务和管理后台三大核心模块，为用户提供从网页浏览到词汇管理的完整学习体验。

### 核心理念

- **i+1 理论**: 内容略高于用户当前水平，既有挑战性又不至于完全看不懂
- **沉浸式学习**: 将整个互联网变成个性化语言学习教材
- **智能辅助**: AI驱动的翻译、发音和词汇管理

### 目标用户

- 外语学习者（英语、日语、韩语等20+种语言）
- 希望在日常浏览中自然提升词汇量的用户
- 需要系统化词汇管理的学习者

## 核心功能模块

## 一、浏览器扩展功能

### 1.1 智能翻译引擎 ⭐

#### 1.1.1 智能语言检测
- **功能描述**: AI自动识别网页源语言，无需用户手动指定
- **支持语言**: 20+种语言（英语、日语、韩语、法语、德语、西班牙语、俄语、意大利语、葡萄牙语、荷兰语、瑞典语、挪威语、丹麦语、芬兰语、波兰语、捷克语、土耳其语、希腊语等）
- **技术实现**: 基于大语言模型的智能文本分析

#### 1.1.2 智能文本处理
- **功能描述**: 使用大语言模型分析网页内容，智能选择适合用户水平的词汇进行翻译
- **核心特性**:
  - 上下文感知的词汇选择
  - 根据用户水平调整词汇难度
  - 考虑语境选择最合适的翻译

#### 1.1.3 精确替换控制
- **翻译比例**: 1%-100%精确控制，支持字符级精确计算
- **用户水平**: 5个级别（初级、初级进阶、中级、中高级、精通）
- **替换策略**: 基于字符数的精确计算，确保翻译比例准确

#### 1.1.4 多API集成
- **OpenAI兼容API**: 支持ChatGPT、Claude、豆包等国产大模型
- **Google Gemini API**: 新增Google Gemini API集成
- **灵活配置**: 自定义API Key、Endpoint、模型名称、Temperature参数
- **多API支持**: 支持配置多个API服务并灵活切换

#### 1.1.5 翻译显示控制
- **翻译位置**: 新增翻译文本位置自定义功能（原文前、原文后、替换原文）
- **括号显示**: 可选择是否显示翻译文本的括号
- **原文显示**: 显示、隐藏或学习模式（模糊效果）显示被翻译的原文

#### 1.1.6 懒加载翻译
- **功能描述**: 滚动到段落时才进行翻译，减少资源消耗和提高性能
- **技术实现**: Intersection Observer API + 智能分段处理

- **实时生成**: 调用AI生成中文词义解释
- **上下文分析**: 支持上下文语境分析，理解更准确
- **智能缓存**: 词义结果本地缓存，减少API调用

#### 1.2.6 快捷键支持

```
一站式学语言助手
├── 浏览器扩展层
│   ├── 智能翻译引擎
│   ├── 发音学习生态系统
│   ├── 视觉体验系统
│   ├── 配置管理系统
│   └── 网站管理功能
├── 后端云服务层
│   ├── 用户认证系统
│   ├── 词汇管理系统
│   ├── 词汇列表系统 (新增)
│   ├── 学习记录系统
│   ├── 复习系统
│   ├── 成就系统 (新增)
│   ├── 目标系统 (新增)
│   ├── 统计分析系统
│   ├── 数据导出系统
│   └── 管理员功能
└── 管理后台层
    ├── 用户仪表板
    ├── 词汇管理
    ├── 词汇列表管理 (新增)
    ├── 复习系统
    ├── 统计分析
    ├── 数据导出
    └── 管理员后台
```

### 技术栈

#### 浏览器扩展
- **框架**: WXT (WebExtension框架)
- **前端**: Vue 3 + TypeScript + Vite
- **UI库**: Tailwind CSS + Lucide Icons + Reka UI
- **API集成**: OpenAI兼容接口 + Google Gemini + Dictionary API + 有道TTS
- **国际化**: Vue I18n支持多语言界面

#### 后端服务
- **框架**: Express.js
- **数据库**: MySQL + Prisma ORM
- **认证**: JWT (Access Token + Refresh Token)
- **验证**: Zod 数据验证
- **API**: RESTful API

#### 管理后台
- **框架**: Vue 3 + TypeScript
- **UI库**: Tailwind CSS + Reka UI
- **状态管理**: Pinia
- **路由**: Vue Router
- **HTTP客户端**: Axios

## 使用场景

### 场景一：日常网页浏览学习
**用户需求**: 在浏览新闻、博客等网页时自然学习外语词汇

**使用流程**:
1. 安装浏览器扩展
2. 配置目标语言和用户水平
3. 正常浏览网页
4. 扩展自动翻译部分词汇
5. 鼠标悬停查看音标和词义
6. 点击播放语音学习发音

### 场景二：系统化词汇管理
**用户需求**: 将学习到的词汇系统化管理，定期复习

**使用流程**:
1. 在浏览过程中收藏感兴趣的单词条目
2. 登录管理后台查看收藏的词汇
3. 创建自定义词汇列表分类管理
4. 使用复习系统定期复习
5. 查看学习统计了解学习进度

### 场景三：多语言学习
**用户需求**: 同时学习多种语言

**使用流程**:
1. 配置多个目标语言
2. 扩展自动检测网页语言
3. 智能翻译到目标语言
4. 分别管理不同语言的词汇
5. 查看各语言的学习统计

### 场景四：Anki集成学习
**用户需求**: 将学习数据导入Anki进行深度学习

**使用流程**:
1. 在管理后台选择导出功能
2. 选择Anki格式导出
3. 下载导出文件
4. 导入到Anki学习软件
5. 使用Anki进行深度学习
